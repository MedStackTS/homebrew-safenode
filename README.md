# homebrew-safenode

Tap Homebrew + binários públicos do **safenode-cli** — CLI local zero-knowledge do SafeNode.

```bash
brew tap medstackts/safenode
brew install safenode
# ou:
brew install medstackts/safenode/safenode
```

Ou via script: `curl -fsSL https://safenode.dev/install.sh | sh`

Os binários são publicados nas Releases (tags `cli/vX.Y.Z`). O código-fonte é privado; aqui ficam só os artefatos públicos.
